# Meetup USQL

A sample Data Lake Analytics project to extract and aggregate captured events from Event Hub in avro/json format.

Source contains one template for Visual Studio (runs on WindowS) and another for Visual Studio Code (runs on macOS, Linux and Windows)

Visual Studio requirements:

- Data Lake Analytics components

Visual Studio Code requirements:

- [Azure Data Lake Tools](https://marketplace.visualstudio.com/items?itemName=usqlextpublisher.usql-vscode-ext)
